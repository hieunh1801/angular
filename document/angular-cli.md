# Angular CLI Usefull script

```bash
# generate new project
ng new todo

# generate new component -> and add to module
ng g c notes-detail # new folder + ts|css|html
ng g c notes-detail --flat # create ts|css|html at current folder

# generate service
ng g s student # create file + markup as root service

# generate module
ng g module admin --routing 
# ./admin/admin.module.ts
# ./app/admin/admin-routing.module.ts
```
